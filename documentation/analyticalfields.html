Analytical fields

Analytical fields define values for spatial coordinates, e.g., for spatially varying material properties.

scalarexpression - Field defined by expression

Relevant module edelweissfe.analyticalfields.scalarexpression

Define a field using a scalar expression.

Options

Option

Type / Default

Description

str (required)

Python expression using variables x, y, z (coordinates); dictionaries contained in model can be accessed

randomscalar - A random field

Relevant module edelweissfe.analyticalfields.randomscalar

Define a random field using the GSTools library. “Müller, S., Schüler, L., Zech, A., and Heße, F.: GSTools v1.3: a toolbox for geostatistical modelling in Python, Geosci. Model Dev., 15, 3161–3182, https://doi.org/10.5194/gmd-15-3161-2022, 2022.”

Options

Option

Type / Default

Description
model

str, default=’Gaussian’

Covariance Model of the spatial random field
mean

float, default=0.0

Mean of the spatial random field
variance

float, default=1.0

Variance of the model
lengthScale

float, default=10.0

Length scale of the model
nu

float, default=1.0

Smoothness parameter for Matern covariance function
seed

int, default=0

Seed of the random number generator
