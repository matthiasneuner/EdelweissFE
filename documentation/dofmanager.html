DOF manager

Module edelweissfe.numerics.dofmanager

This module contains important classes for describing the global equation system by means of a sparse system.

class edelweissfe.numerics.dofmanager.DofManager(nodeFields, scalarVariables=None, elements=None, constraints=None, nodeSets=None, initializeVIJPattern=True, initializeAccumulatedNodalFluxesFieldwise=True, determiningIndexToHostObjectMappping=True)[source]

The DofManager

  • analyzes the domain (nodes and constraints),

  • collects information about the necessary structure of the degrees of freedom

  • handles the active fields on each node

  • counts the accumulated number of associated elements on each dof (for the Abaqus like convergence test)

  • supplies the framework with DofVectors and VIJSystemMatrices

Parameters:

  • nodeFields (list[NodeField]) – The list of NodeFields which should be represented in the DofVector structure.

  • scalarVariables (list[ScalarVariable]) – The list of ScalarVariables which should be represented in the DofVector structure.

  • elements (list[BaseNodeCouplingEntity]) – The list of Elements for which a map to the respective indices should be created.

  • constraints (list[BaseNodeCouplingEntity]) – The list of Constraints for which a map to the respective indices should be created.

  • nodeSets (list[NodeSet]) – The list of NodeSets for which a map to the respective indices should be created.

  • initializeVIJPattern (bool) – Whether to initialize the VIJ pattern (I and J vectors) during construction. Can be set to False if the pattern will be initialized later or not needed.

  • initializeAccumulatedNodalFluxesFieldwise (bool) – Whether to compute the accumulated nodal fluxes fieldwise during construction. This is needed for the Abaqus like convergence test, but can be set to False if not needed.

  • determiningIndexToHostObjectMappping (bool) – Whether to determine the mapping from indices in the DofVector to their host objects (e.g., Nodes) during construction. This can be set to False if the mapping will be determined later or not needed.
